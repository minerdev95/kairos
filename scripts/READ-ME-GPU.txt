KAIROS 0.1.0 — Windows GPU build (NVIDIA CUDA)
==============================================

This build adds KAIROS's own GPU mining kernels. It mines Ergo (ERG / Autolykos2)
on your NVIDIA GPU with KAIROS's own engine — no third-party miner.

Requirements
------------
- An NVIDIA GPU (GeForce/RTX etc.).
- A reasonably recent driver: CUDA 12.5-class or newer
  (NVIDIA driver 555.85+). Update via GeForce Experience if in doubt.
- cudart64_12.dll ships in this folder next to kairos.exe — keep them together.

Prove the GPU kernel is correct on YOUR card (optional, recommended)
-------------------------------------------------------------------
  kairos erg-selftest
It recomputes Ergo's official test vector on your GPU and checks it byte-for-byte.
A "GPU kernel matches the official Ergo KAT" line means you're good to mine.

Mine Ergo (ERG)
---------------
  kairos mine autolykos2 stratum+tcp://POOL:PORT  9YOUR_ERGO_ADDRESS.rig1  --yes

  e.g.  kairos mine erg stratum+tcp://de.ergo.herominers.com:1180 9hYourAddr.rig1 --yes

Notes
-----
- "erg" is an alias for "autolykos2". Any Ergo pool works — just change the URL.
- Ergo's dataset is ~6 GB; an 8 GB card fits it but is near the limit. 10-12 GB+
  cards have more headroom.
- Nothing mines without your explicit --yes.
- A disclosed 1% developer time-slice supports development (see README / LICENSE).

The GUI and all CPU/ASIC features from the standard build are included here too.
